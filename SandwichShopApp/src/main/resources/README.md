# 🥪 DELI-CIOUS Sandwich Shop


## Overview

#### This is a Java console-based Sandwich Shop Ordering System that allows customers to create custom sandwich orders, add drinks and chips checkout through an interactive menu-driven interface.

#### The project demonstrates:

#### Object-Oriented Programming (OOP)  

#### Console input/output  

#### Menu handling  

#### Input validation  

#### Order calculations

## 📌 Features

Create a new order
Build custom sandwiches:
Choose bread type       
Choose sandwich size (4", 8", 12")  
Add meats, cheeses, toppings, sauces, and sides  
Optional toasted sandwich   
Add drinks (size + flavor)    
Add chips (flavor)    
Checkout system with receipt generation   
Save receipt to file   
Input validation for all menu selection
